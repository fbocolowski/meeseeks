# SSSM - Stupid Simple Server Monitor

## Requirements

- MongoDB 3.4
- Ruby 2.5.3
- Rails 5.2.2 or above

## Development

Clone the repository and install dependencies running ```bundle install```.

Start the server running ```rails server```.

Go to http://localhost:3000.
